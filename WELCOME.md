## 📅 YOUR WEEK 1 MISSION

### Task 1: Get Images (1-2 hours)
Find 6 simple images:
- 4 product photos (t-shirt, hoodie, jersey, polo)
- 2 collection lifestyle photos
- Free from Unsplash.com, Pexels.com, or use your own
- See: [IMAGE_REQUIREMENTS.md](IMAGE_REQUIREMENTS.md)

### Task 2: Setup Project (20 minutes)
```bash
npm create vite@latest frontend -- --template react
cd frontend && npm install
npm install -D tailwindcss postcss autoprefixer lucide-react axios
npx tailwindcss init -p
```

### Task 3: Copy Files (5 minutes)
Copy provided files into your React project

### Task 4: Add Your Images (30 minutes)
Replace image placeholders with your URLs

### Task 5: Run It! (1 minute)
```bash
npm run dev
```
See your store at http://localhost:5173 ✨
